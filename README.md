# Aronaro Labs LTD — Website

This repository contains assets, notes, and implementation references for the Aronaro Labs LTD website.

> Primary objective: a conversion-led website that generates enquiries for UK SMEs.

---

## Core goals

- Clear offer and positioning (lead-generation websites)
- Mobile-first layout and fast load speeds
- Simple conversion path (CTA → contact → booked call)
- Strong SEO foundations (local + service intent)
